# Java String Exercises

A small Java project demonstrating fundamental **String manipulation techniques**.  
Each exercise focuses on a specific method from the Java `String` API and reinforces core programming concepts.

---

## 📖 Project Overview

This project contains a series of structured exercises designed to practice:

- String length calculation
- Character indexing
- Substring extraction
- Case conversion
- Whitespace trimming
- Type conversion
- String splitting
- Working with character arrays

The goal is to strengthen understanding of Java fundamentals and core String operations.

---

## 🧪 Implemented Exercises

1. **Length of a String**  
   Retrieve the length of `"Java"` using `length()`.

2. **Character at Index**  
   Access a character at a specific index using `charAt()`.

3. **Index Position of Character**  
   Find the position of `'o'` using `indexOf()`.

4. **Substring Extraction**  
   Extract `"not as long"` using `substring()`.

5. **Lowercase & Uppercase Conversion**  
   Convert text using `toLowerCase()` and `toUpperCase()`.

6. **Trim String**  
   Remove leading/trailing whitespace using `trim()`.

7. **Parse to String**  
   Convert an integer to a `String` and concatenate values.

8. **Split String**  
   Split text using `split()` and store results in arrays.

9. **Convert String to Char Array**  
   Use `toCharArray()` and iterate through elements.

10. **Convert Char Array to String**  
    Convert `char[]` back into a `String`.
